Kebijakan Privasi Buku Zinuno AFY

Terakhir Diperbarui: 7 April 2026

Sinode AFY ("kami") mengelola aplikasi mobile Buku Zinuno AFY. Halaman ini menginformasikan kebijakan kami mengenai pengumpulan, penggunaan, dan pengungkapan data pribadi saat Anda menggunakan aplikasi kami.

1. Pengumpulan dan Penggunaan Informasi

Kami berkomitmen untuk melindungi privasi jemaat dan pengguna aplikasi kami.

  • Data Pribadi: Aplikasi ini tidak mewajibkan pendaftaran akun. Kami tidak mengumpulkan data identitas pribadi seperti nama, alamat email, atau nomor telepon Anda.
  • Data Lokal: Pilihan "Lagu Favorit" yang Anda simpan disimpan secara lokal di perangkat Anda menggunakan Room Database. Kami tidak memiliki akses ke daftar lagu favorit Anda.

2. Layanan Pihak Ketiga (Firebase)

Untuk memberikan pengalaman terbaik, aplikasi ini menggunakan layanan dari pihak ketiga untuk sinkronisasi data:

  • Google Firebase: Kami menggunakan Firebase Realtime Database hanya untuk memperbarui lirik lagu, informasi aplikasi, dan versi data secara otomatis. Firebase mungkin mengumpulkan informasi teknis tertentu (seperti ID perangkat atau log sistem) untuk tujuan analitik dasar dan kinerja aplikasi sesuai dengan Kebijakan Privasi Google.

3. Keamanan Data

Keamanan data Anda adalah prioritas kami. Karena kami tidak mengumpulkan data sensitif, risiko kebocoran data pribadi sangat minimal. Namun, perlu diingat bahwa tidak ada metode transmisi melalui internet atau metode penyimpanan elektronik yang 100% aman.

4. Iklan dan Konten Sensitif

  • Tanpa Iklan: Aplikasi ini sepenuhnya bebas dari iklan pihak ketiga untuk menjaga kekhusyukan ibadah.
  • Konten: Semua konten dalam aplikasi ini adalah lirik lagu rohani dan informasi resmi dari Sinode AFY.

5. Izin Perangkat

Aplikasi mungkin meminta izin akses internet untuk:

  • Melakukan sinkronisasi lirik lagu terbaru dari server Firebase.
  • Membuka tautan ekspor (WhatsApp/Email) saat Anda menggunakan fitur "Hubungi Kami".

6. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi kami dari waktu ke waktu. Kami akan memberi tahu Anda tentang perubahan apa pun dengan memperbarui nomor versi sinkronisasi (app_info) di dalam aplikasi.
